{% with terms=lang|default:'fr' %} {% if lang == 'fr' %}

🔓 Phishing

Tentative d'obtenir des informations sensibles (mots de passe, données bancaires) en se faisant passer pour une entité de confiance.

🛡️ Ransomware

Logiciel malveillant qui chiffre vos données et exige une rançon pour les déchiffrer.

🔍 Pentesting

Test d'intrusion éthique : simulation d'une cyberattaque pour identifier les vulnérabilités d'un système.

🌐 Firewall

Système de sécurité réseau qui filtre le trafic entrant et sortant selon des règles prédéfinies.

🔐 2FA / MFA

Authentification à plusieurs facteurs : une couche de sécurité supplémentaire au-delà du mot de passe.

📋 ISO 27001

Norme internationale pour la gestion de la sécurité de l'information (SMSI).

🕵️ OSINT

Open Source Intelligence : collecte et analyse d'informations provenant de sources publiques.

⚡ SOC / SIEM

Security Operations Center : centre de surveillance et de réponse aux incidents de sécurité.

🔒 Chiffrement

Processus de conversion de données en code secret pour empêcher l'accès non autorisé.

{% elif lang == 'de' %}

🔓 Phishing

Versuch, an sensible Informationen zu gelangen, indem man sich als vertrauenswürdige Instanz ausgibt.

🛡️ Ransomware

Schadsoftware, die Daten verschlüsselt und Lösegeld für die Entschlüsselung fordert.

🔍 Pentesting

Ethischer Penetrationstest: Simulation eines Cyberangriffs zur Identifizierung von Schwachstellen.

🌐 Firewall

Netzwerksicherheitssystem, das den ein- und ausgehenden Verkehr filtert.

🔐 2FA / MFA

Multi-Faktor-Authentifizierung: zusätzliche Sicherheitsebene über das Passwort hinaus.

📋 ISO 27001

Internationale Norm für Informationssicherheits-Managementsysteme (ISMS).

🕵️ OSINT

Open Source Intelligence: Sammlung und Analyse von Informationen aus öffentlichen Quellen.

⚡ SOC / SIEM

Security Operations Center: Überwachungs- und Reaktionszentrum für Sicherheitsvorfälle.

🔒 Verschlüsselung

Umwandlung von Daten in Geheimcode, um unbefugten Zugriff zu verhindern.

{% else %}

🔓 Phishing

Fraudulent attempt to obtain sensitive information by impersonating a trustworthy entity.

🛡️ Ransomware

Malware that encrypts your data and demands payment for decryption.

🔍 Pentesting

Ethical penetration testing: simulating cyberattacks to identify system vulnerabilities.

🌐 Firewall

Network security system that monitors and filters incoming and outgoing traffic.

🔐 2FA / MFA

Multi-Factor Authentication: additional security layer beyond passwords.

📋 ISO 27001

International standard for information security management systems (ISMS).

🕵️ OSINT

Open Source Intelligence: gathering and analyzing information from public sources.

⚡ SOC / SIEM

Security Operations Center: centralized security monitoring and incident response.

🔒 Encryption

Process of converting data into secret code to prevent unauthorized access.
