{% if lang == 'fr' %}C'est quoi un pentest ?{% elif lang == 'de' %}Was ist ein Pentest?{% else %}What is a pentest?{% endif %} ▼

{% if lang == 'fr' %}Un test d'intrusion (pentesting) est une simulation d'attaque cyber réalisée par un expert autorisé. L'objectif est d'identifier les vulnérabilités avant que de vrais attaquants ne les exploitent. Cela inclut des tests de réseau, d'applications web, de sécurité sans fil et d'ingénierie sociale.{% elif lang == 'de' %}Ein Penetrationstest (Pentesting) ist eine simulierte Cyberattacke durch einen autorisierten Experten. Ziel ist es, Schwachstellen zu identifizieren, bevor echte Angreifer sie ausnutzen. Dies umfasst Netzwerktests, Webanwendungstests, WLAN-Sicherheitstests und Social Engineering.{% else %}A penetration test (pentesting) is a simulated cyber attack by an authorized expert. The goal is to identify vulnerabilities before real attackers exploit them. This includes network testing, web application testing, wireless security testing, and social engineering.{% endif %}

{% if lang == 'fr' %}Quelle est la différence entre un virus, un ver et un cheval de Troie ?{% elif lang == 'de' %}Was ist der Unterschied zwischen einem Virus, einem Wurm und einem Trojaner?{% else %}What's the difference between a virus, a worm, and a Trojan?{% endif %} ▼

{% if lang == 'fr' %}Un virus s'attache à un fichier et se propage quand ce fichier est exécuté. Un ver (worm) se propage automatiquement via le réseau sans nécessiter d'interaction humaine. Un cheval de Troie (Trojan) se fait passer pour un logiciel légitime pour tromper l'utilisateur. Les trois sont des malwares, mais leurs méthodes de propagation diffèrent.{% elif lang == 'de' %}Ein Virus heftet sich an eine Datei und verbreitet sich, wenn diese Datei ausgeführt wird. Ein Wurm verbreitet sich automatisch über das Netzwerk ohne menschliches Zutun. Ein Trojaner gibt sich als legitime Software aus, um den Benutzer zu täuschen. Alle drei sind Malware, aber ihre Verbreitungsmethoden unterscheiden sich.{% else %}A virus attaches to a file and spreads when that file is executed. A worm spreads automatically over the network without human interaction. A Trojan horse disguises itself as legitimate software to trick the user. All three are malware, but their propagation methods differ.{% endif %}

{% if lang == 'fr' %}La DSGVO s'applique-t-elle à ma petite entreprise ?{% elif lang == 'de' %}Gilt die DSGVO für mein kleines Unternehmen?{% else %}Does GDPR apply to my small business?{% endif %} ▼

{% if lang == 'fr' %}Oui ! La DSGVO s'applique à toute organisation traitant des données personnelles de citoyens européens, quelle que soit sa taille. Les sanctions peuvent atteindre 20 millions € ou 4% du chiffre d'affaires annuel mondial. Même les petites structures doivent tenir un registre des traitements et sécuriser les données.{% elif lang == 'de' %}Ja! Die DSGVO gilt für jede Organisation, die personenbezogene Daten von EU-Bürgern verarbeitet, unabhängig von ihrer Größe. Die Strafen können bis zu 20 Mio. € oder 4% des weltweiten Jahresumsatzes betragen. Auch kleine Unternehmen müssen ein Verarbeitungsverzeichnis führen und Daten schützen.{% else %}Yes! GDPR applies to any organization processing personal data of EU citizens, regardless of size. Penalties can reach €20 million or 4% of annual global turnover. Even small businesses must maintain a processing register and secure data.{% endif %}

{% if lang == 'fr' %}Qu'est-ce que la norme ISO 27001 ?{% elif lang == 'de' %}Was ist die ISO 27001-Norm?{% else %}What is ISO 27001?{% endif %} ▼

{% if lang == 'fr' %}L'ISO 27001 est une norme internationale pour la gestion de la sécurité de l'information (SMSI). Elle définit les exigences pour établir, mettre en œuvre, maintenir et améliorer un système de management de la sécurité de l'information. Être certifié ISO 27001 démontre un engagement sérieux envers la sécurité.{% elif lang == 'de' %}ISO 27001 ist eine internationale Norm für Informationssicherheits-Managementsysteme (ISMS). Sie definiert die Anforderungen für die Einrichtung, Umsetzung, Aufrechterhaltung und Verbesserung eines Informationssicherheits-Managementsystems. ISO 27001-zertifiziert zu sein, zeigt ein ernsthaftes Engagement für Sicherheit.{% else %}ISO 27001 is an international standard for information security management (ISMS). It defines requirements for establishing, implementing, maintaining, and improving an information security management system. Being ISO 27001 certified demonstrates a serious commitment to security.{% endif %}
